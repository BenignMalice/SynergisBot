▶ WHAT THIS DOES
  Allows ChatGPT to execute trades directly on your MT5 account via HTTP API

▶ QUICK START (3 Steps)
  
  1. START THE SERVER
     Double-click: start_with_ngrok.bat
     
  2. VERIFY IT'S RUNNING
     Open browser: http://localhost:8000/docs
     
  3. CONFIGURE CHATGPT
     - Go to ChatGPT → Settings → GPT Builder
     - Create new GPT → Add Action
     - Import: openai.yaml (FULL API - Now Implemented!)
     - Save

▶ TEST COMMANDS (Ask ChatGPT)
  
  "Check my MT5 account balance"
  "Analyze XAUUSD and recommend a trade"
  "What patterns do you see in BTCUSD?"
  "Should I exit my XAUUSD position?"
  "Execute a BUY trade on BTCUSD with SL at 115000 and TP at 125000"
  "What's the price prediction for XAUUSD?"

▶ API ENDPOINTS (16 Total!)
  
  POST /mt5/execute           → Execute a trade directly
  POST /signal/send           → Send signal to Telegram
  GET  /api/v1/account        → Get account info
  GET  /api/v1/symbols        → List symbols
  GET  /health                → Check status
  GET  /market/analysis/{sym} → Market analysis
  GET  /risk/metrics          → Risk metrics
  GET  /monitor/status        → Monitoring status
  ... and 8 more endpoints!

▶ TROUBLESHOOTING
  
  Problem: "Tool call not successful"
  Solution: Make sure both API server and ngrok are running
  
  Problem: "MT5 not connected"
  Solution: Open MT5 terminal and log in
  
  Problem: Can't access http://localhost:8000
  Solution: Restart API server, check for port conflicts
  
  Problem: ChatGPT says "I can't access that"
  Solution: Check ngrok is running at verbally-faithful-monster.ngrok-free.app
